## Loan Application API

Core features:
- Creates loan applications
- Reads loan applications
- Update loan applications
- Delete loan applications

Key endpoints:

- POST /loans
- GET /loans
- GET /loans/{loan_id}
- PUT /loans/{loan_id}
- PATCH /loans/{loan_id}
- DELETE /loans/{loan_id}

Stack:
- Python
- FastAPI 
- In-memory storage
- No front-end

The objective of developing this API is to learn the fundamentals of REST APIs.
